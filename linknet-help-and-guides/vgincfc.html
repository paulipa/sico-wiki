SiCo VG FC Trainee Checklist 

Tagging. the role of the FC will be to tag targets for the rest of the fleet to follow:

 - for Snipers, kill priority is: Mara/Niarja -> Auga -> Deltole/Romi 

 - for DPS, kill priority is: Tama -> Schmaeel -> Renyn -> Eystur 

(those with a '/' between them never spawn together, so they have the same priority)

Exceptions: 

 - in OTAs, Eysturs respawn -> Tag 'X'

 - in OTAs, NM-heavy fleet can snipe up to 3 Tamas in wave 1+2 before they burn in 

THESE MUST GET TAGGED A,B,C IN SEQUENCE TO BREAK REPS

 - in NCOs, 3rd wave spawns Sansha Commander -> primary Tag '1'

 - in NMCs, wave 2+3 have Tamas that burn in and can be sniped by Nightmares until they get to 35km

No need for special tags, just call for nightmares to shoot them

Last wave identification:

  -NCO: Last wave has only frigates 

 - NMC: Last wave spawns Auga + Romi (2nd to last only has Romi) 

 - OTA: Last wave spawns close (35km) and above grid

When to broadcast align/drones in:

 - NCO: When there are 5 frigs left 

 - NMC: When there are 2 cruisers left (2 Romis, but can be 1 Auga + 1 Romi) 

 - OTA: When starting to shoot Deltole in last wave

Exceptions:

 - In OTAs, when fleet is light on DPS and last wave is heavy on Tamas (4+), tell Snipers to shoot webbed Tamas before Deltole and don’t broadcast until Tamas are dead.

-OTA: When starting to shoot Deltole in last wave
